Help Page



"Re-load Summary" button:

Save current selections and annotations and redisplay the summary page. Use this option to display new sequence versions arrising from edits in the alignment view. Always reload the the summary view and check selections before saving assignments and quitting.

"View Unusual Linkages" button:

Display a table of atypically long polymer linkages.

"Save Assignments" button:

Saves current selections and annotation results in an assignment file and the merges these assignments into the current model file.

"Quit for now" button:

End the current session without updating assignment and model files in the archive directory.

This option will save current selections that will be reused if the module is restarted from the workflow manager UI interface.

"All Done" button:

End the current session and update the archive with any save assignments.

This option does not save assignments or create a new model file. You must click on the "Save" button to do so.

"Save in-progress sequence annotation to archive" button:

This option will save current selections into archive directory. The saved information will be reused if a new sequence workflow is started from the workflow manager.

"Remove in-progress sequence annotation from archive" button:

This option will remove the saved current selections from archive directory.

"SELECT" check-box:

Selects this the sequence and the associated annotations for export. Data for selected sequences are stored in the sequence assignment file upon pressing the "Save assignments" button.

"ALIGN ALL" check-box:

Selects all sequences in the group for alignment.

"Edit Sequence/Partition/Taxonomy Source" button:

Displays an input form with options to edit taxonomy and applicable residue ranges for the entity.

An alternative one-letter code sequence with parenthetical modifications may be entered for the entity sequence.

An option is provided to consolidate all of the instances from another entity into the current entity. The entity without instances is effectively deleted and will not appear in the updated model file.

A new reference sequence database search may optionally follow taxonomy and partition changes.

"Edit Source Details" button:

Displays an input form with options to edit various enity and source annotations

"Load Reference Sequence" button:

Displays an input form with options to input a new reference sequence or edit the related residue range

To edit a single residue:

Edit one residue at a time
double-click on residue, then edit in place
[ENTER] or [RETURN] to save the edited changes
[ESC] aborts any edit in progress

To move/shift a single residue:

Click on 'Activate Move' button + left-click on residue, then shift the residue to an adjacent position

To delete residue(s):

  1. Single residue
    a). Use [SHIFT] + left-click on residue, then edit in place by backspace to clear the letter symbol
    b). Left-click on a single residue and drag cursor over just that residue, then click on "Mark for Deletion" to highlight the residue in purple.
  2. Range of residues:
    Left-click on single residue and drag cursor over range of adjacent residues, releasing left mouse button when done highlighting the desired range. Then click on "Mark for Deletion" which will highlight the residues in purple.

To commit edits:

Click on "Save Alignment" button in order to commit any edits to an output file.
Successive clicks on "Save Alignment" button creates a new version of the output file.

To incrementally undo any edits:

Clicking on the "Undo" button will undo the most recent edit action. Such that if there were a series of edit actions, successive clicks on "Undo" will rollback each of these edits from most recent to least recent for all edit actions undertaken since the beginning of the editing session or since the "Save Alignment" button was last pressed.
