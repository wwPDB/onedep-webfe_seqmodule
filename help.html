Help Page



"Re-load Summary" button:

Save current selections and annotations and re-display the summary page. Use this option to display new sequence versions arising from edits in the alignment view. Always reload the the summary view and check selections before saving assignments and quitting.

"View Unusual Linkages" button:

Display a table of atypically long polymer linkages.

"Save Assignments" button:

Saves current selections and annotation results in an assignment file and merges these assignments into the current model file. Please also note that upon clicking "Save Assignments" button, the saved in-progress sequence annotation is also removed by default.

"Quit for now" button:

End the current session without updating assignment and model files in the archive directory.

This option will save current selections and annotation if the module is re-opened. Use this button if you want to go home now and resume sequence annotation immediately tomorrow.

"All Done" button:

End the current session and update the archive with any save assignments.

This option does not save assignments or create a new model file. You must click on the "Save Assignments" button to do so.

"Save in-progress sequence annotation to archive" button:

This option will save the current selections and annotation into archive directory. The saved information will be reused if sequence module is re-started from the work flow manager. Use this button if you want to save current sequence annotation, but also need to update model file (through other modules or editor), and then re-run sequence module to finish sequence assignment.

"Remove in-progress sequence annotation from archive" button:

This option will remove the saved current selections from archive directory. Please also note that upon clicking "Save Assignments" button, the saved in-progress sequence annotation is also removed by default.

"SELECT" check-box:

Select the sequence and the associated annotations for export. Data for selected sequences are stored in the sequence assignment file upon pressing the "Save Assignments" button.

"ALIGN ALL" check-box:

Select all sequences in the group for alignment.

"Edit Sequence/Partition/Taxonomy Source" button:

Displays an input form with options to edit taxonomy and applicable residue ranges for the entity.

An alternative one-letter code sequence with modified residue in parenthesis may be entered for the entity sequence. Alignment will be re-generated upon submitting the new sequence. It is the most convenient way to add tags at the termini.

For chimera protein, please split the sequence by absolute range and provide appropriate source tax id.

An option of "Add instances from entity" at the bottom is provided to consolidate all of the instances from another entity into the current entity. The entity without instances is effectively deleted and will not appear in the updated model file.

A new reference sequence database search may optionally follow taxonomy and partition changes.

"Edit Source Details" button:

Displays an input form with options to edit various entity and source annotations.

"Load Reference Sequence" button:

Displays an input form with options to input a new reference sequence or edit the related residue range (in UNP/GB number).

To edit a single residue:

Edit one residue at a time
double-click on residue, then edit in place
[ENTER] or [RETURN] to save the edited changes
[ESC] aborts any edit in progress

To move/shift a single residue:

Click on 'Activate Move' button + left-click on residue, then shift the residue to a new position

To move/shift blocks of residues:

  1. Click the "Make Block Move" button.
  2. Specify the start and end positions of the block to be moved, and specify the block's "move to" position. The "move to position" is the destination of the starting residue of the block if the block is being moved to the left, and is the destination of the end residue of the block if the block is being moved to the right. All numbers entered should be positions within the polymer sequence (i.e., the top row of the alignment) and not residue numbers from the model or sequence database reference(s).
  3. When you have entered the numbers, you can click the "Check & Display Selections" button to see both the selected block and its "move to" position highlighted in orange.
  4. When you ready to move a block, click the "Move" button. The "Undo" button can be used if necessary.

To delete residue(s):

  1. Single residue:
    a). Double click on residue, then edit in place by backspace to clear the letter symbol
    b). Left-click on a single residue and drag cursor over just that residue, then click on "Mark for Deletion" to highlight the residue in purple. The residue will be highlighted in purple. Then click "Save Alignment" on alignment page to realize the deletion.
  2. Range of residues:
    Left-click on single residue and drag cursor over range of adjacent residues, releasing left mouse button when done highlighting the desired range. Then click on "Mark for Deletion" which will highlight the residues in purple. Then click "Save Alignment" on alignment page to realize the deletion.

To commit edits:

Click on "Save Alignment" button in order to commit any edits to an output file, which will create a new version of the output file.

To incrementally undo any edits:

Clicking on the "Undo" button will undo the most recent edit action. Such that if there were a series of edit actions, successive clicks on "Undo" will rollback each of these edits from most recent to least recent for all edit actions undertaken since the beginning of the editing session or since the "Save Alignment" button was last pressed.
